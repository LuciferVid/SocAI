HOW IT WORKS

Three stages. Zero manual effort.

Every log that enters your system is processed through a purpose-built ML pipeline in under 30 milliseconds.

01
⚡

Ingest

Network sensors stream logs via Redis Pub/Sub. Every HTTP request, SSH attempt, and DNS query is captured and parsed in real-time — no batching, no delay.

02
🧠

Analyze

A feature engine extracts 5 behavioral signals per event using Redis Sorted Sets. Two ML models — Isolation Forest + Autoencoder — score every request for anomaly probability.

03
🚨

Respond

Threats that exceed the confidence threshold instantly fire alerts via webhooks, update IP reputation scores, and push to the live WebSocket dashboard.

CAPABILITIES

Everything a SOC needs. Nothing it doesn't.

Built for real threats. Every feature earns its place.

DETECTION

Hybrid ML Engine

Isolation Forest finds statistical outliers. Autoencoder catches patterns never seen before. Combined scoring reduces false positives by 60%.

INTELLIGENCE

IP Reputation

EWMA-based scoring tracks every source IP over time. Suspicious actors are flagged, escalated, and context is preserved across events.

CLASSIFICATION

Attack Labeling

Brute force hammering, DDoS spikes, suspicious endpoint crawling — each detected pattern gets a human-readable attack label.

ALERTING

Instant Notifications

Score crosses the threshold? Alert fires in under 30ms via webhook or email. Configurable cooldown prevents alert fatigue.

LEARNING

Self-Improving

Label false positives. Trigger retraining via API. Models hot-swap on disk — zero downtime, always improving.

VISIBILITY

Live Dashboard

WebSocket-powered event stream. Real-time charts. Severity-coded alert feed. Full drill-down into any event.

ARCHITECTURE

Intentional design.
Every decision earned.

No over-engineering. No enterprise bloat. A lean, fast system built to do one thing extremely well.

Redis Pub/Sub
Ultra-low latency event bus. Fits in 512MB RAM. No Zookeeper, no cluster management.
Sorted Sets
O(log N) sliding-window feature queries. No cron jobs, no external schedulers.
Hybrid ML
Deterministic rules catch known attacks. Neural models discover the unknown.
EWMA Scoring
Exponentially weighted reputation. Recent events matter more. One mistake doesn't stain forever.
Hot-swap
Retrain models without restarting the service. Scorer watches artifacts on disk.
