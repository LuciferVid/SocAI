THE PIPELINE

Three stages. Zero manual configuration.

Every log is processed through a specialized ML pipeline and scored for threat probability in under 3 milliseconds.

01
⚡

Ingest

Logs stream in real-time via Redis Pub/Sub. Every HTTP request, SSH attempt, and DNS query is captured, parsed, and fed into the detection engine instantly.

02
🧠

Analyze

A feature engine extracts behavioral signals from each event using sliding-window aggregations. Two ML models — Isolation Forest (statistical outliers) + Autoencoder (learned patterns) — score the anomaly probability.

03
🚨

Respond

Threats crossing the confidence threshold instantly fire alerts, update IP reputation, and push to the WebSocket dashboard. Full drill-down context available for investigation.

Hybrid ML Scoring

Isolation Forest catches statistical anomalies. Autoencoder learns complex patterns. Combined results reduce false positives by 60%.

INTELLIGENCE

IP Reputation

EWMA-based scoring tracks every source IP over time. Suspicious actors get flagged. Context preserved across all events.

CLASSIFICATION

Attack Labeling

Brute force hammering, DDoS spikes, path traversal — each threat pattern gets a human-readable classification label.

ALERTING

Score thresholds crossed? Alert fires in under 3ms via webhooks or email. Configurable cooldowns prevent alert fatigue.

LEARNING

Hot-Swap Models
