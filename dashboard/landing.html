PIPELINE

From raw log to fired alert

Six stages. All streaming. Sub-30ms end to end.

1
Simulated Logs Traffic generator pumps events continuously
2
Redis Pub/Sub Events published to channel. Ultra-low latency.
3
Worker Subscribes, parses IP / method / path / status
4
Feature Engine Redis Sorted Sets build 5 real-time features
5
ML Scorer Isolation Forest + Autoencoder score each event
6
Alert + Finish Threshold breach → webhook fired + WS push
PHASE 01

Isolation Forest

Unsupervised anomaly scoring — isolates outliers in multidimensional feature space. No labelled data required.

PHASE 02

Autoencoder

Neural reconstruction error — high reconstruction error signals a pattern the network has never seen before.

PHASE 03

Hybrid Engine

Deterministic rules for brute force and DDoS combined with weighted ML anomaly scores for discovery detection.

CAPABILITIES

Built for real threats

Not a toy. Every design decision made for production-grade security monitoring.

Real-time streaming

Redis Pub/Sub ingestion. Every log scored the moment it arrives. No batch delays.

IP reputation system

EWMA scoring tracks each IP dynamically. Suspicious IPs auto-tagged and escalated.

Attack classification

Brute force, DDoS spikes, and suspicious endpoint patterns detected and labelled.

Instant alerts

Anomaly score above threshold fires a webhook or email alert in under 30ms.

Self-learning loop

Label false positives. Trigger retraining. Models hot-swap without service restart.

Live SOC dashboard

WebSocket event stream. Real-time charts. Alert feed and IP drill-down built in.

ARCHITECTURE

Every decision made intentionally

Redis Pub/Sub
Ultra-low latency ingestion. Fits free-tier cloud (512MB RAM). No Zookeeper overhead.
Redis Sorted Sets
O(log N) sliding-window feature queries without external schedulers or cron jobs.
Hybrid ML + Rules
Deterministic rules catch known attacks; ML isolates novel, never-seen anomalies.
EWMA Reputation
Recent events weighted more heavily. One false positive won't stain an IP forever.
Hot-swap Models
Scorer reloads artifacts on disk. No restart, no downtime during retraining.
